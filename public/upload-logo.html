Logo Upload Instructions

How to Add Your Custom Logo

  1. Prepare your logo image file (PNG, JPG, or SVG format recommended)
  2. Rename your image file to custom-logo.png (or .jpg/.svg depending on your file type)
  3. Copy your renamed logo file to the /public directory of this project
  4. Replace the existing placeholder file
  5. Restart the development server if needed
Note: The website is already configured to use a file named custom-logo.png in the public directory. If your image has a different format (like JPG or SVG), you'll need to update the references in Navigation.tsx and index.html to match your file extension.

Recommended Logo Specifications

  • Format: PNG with transparency or SVG (vector) for best quality
  • Size: At least 200x200 pixels
  • Aspect ratio: Square or slightly rectangular
  • File size: Keep under 200KB for optimal loading speed

After adding your logo, it will appear in the navigation bar and as the favicon in browser tabs.
